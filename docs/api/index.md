## Wrappers (Gymnax compatibility)

The ``octax.wrappers`` module provides ``OctaxGymnaxWrapper`` for Gymnax compatibility. It is documented in the main [README](https://github.com/riiswa/octax#gymnax-integration). Requires the optional dependency ``gymnax``.
